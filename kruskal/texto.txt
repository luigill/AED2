Algoritmo de Kruskal

- Cada vértice é uma árvore independente (floresta).
- A cada iteração, procure a aresta de "menor peso" que conecte duas árvores diferentes.
- Os vértices das árvores selecionadas passam a fazer parte de uma mesma árvore. Repita o processo.
- O procesos termina quando todos os vértices fazem parte de uma mesma árvore ou quando não
 se pode encontrar uma aresta que satisfaça a condição.
